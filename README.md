
# 📚 RAG Chatbot

A Retrieval-Augmented Generation (RAG) chatbot built using LangChain, FAISS, Sentence Transformers, and Google Gemini.

## Features

* PDF document ingestion
* Text chunking
* Embedding generation
* FAISS vector database
* Semantic similarity search
* Question answering using Gemini

## Workflow

PDF → Chunking → Embeddings → FAISS → Retrieval → Gemini → Answer

## Technologies Used

* Python
* LangChain
* FAISS
* Sentence Transformers
* Google Gemini
* PyPDF

## Installation

```bash
pip install -r requirements.txt
```
